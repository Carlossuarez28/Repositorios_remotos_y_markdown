### Estructuras de Control
Las estructuras de control nos permiten decidir el flujo de ejecución de nuestro código:

1.  **Condicionales**: Permiten tomar decisiones (`if`, `else`, `switch`).
2.  **Bucles (Ciclos)**: Repiten bloques de código bajo ciertas condiciones:
    * Bucles definidos:
        * `for`: Ideal cuando conocemos el número de iteraciones.
    * Bucles indefinidos:
        * `while`: Se ejecuta mientras una condición sea verdadera.
        * `do-while`: Asegura al menos una ejecución.
